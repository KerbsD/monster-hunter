Great Sword

Great Sword (大剣 dai ken, "big sword") is one out of the fourteen weapon categories in Monster Hunter Rise (MHR or MHRise). Like all Weapons, it features a unique set of moves and an upgrade path that branches out depending on the materials used.

Hammer

Hammer

The Hammer (ハンマー Hanmā) weapon class is a fairly common weapon type able to be used by players since the first Monster Hunter game. Staunch and resilient, the hammer is perhaps the first representative weapon-of-choice to dictate pure, unadulterated damage directly to the target.
